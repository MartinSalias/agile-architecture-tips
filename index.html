Welcome to Agile Architecture Tips

This is a companion website for the Agile Architecture book.

Technology Details over time

While the book is expected to be useful for several years, detailed descriptions of specific technologies like platforms, tools, languages and frameworks will surely change over time. One of the main goals of this website is to provide more detailed descriptions which can be linked from the book, but can also be updated as things evolve.

Alternative Techniques

After the book reach its final stage (at least for a given edition), new ideas and techniques will surely come to mind and become important, so this website will provide extensions to the book chapters with some of these features.

Open Debate

The final goal of this website is to provide space for feedback and discussion about the book, also linked to specific sections and chapters.
